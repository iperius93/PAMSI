Detailed Description

Modeluje pojęcie kwadratu.

Klasa modeluje pojęcie kwadratu o zadanej długości boku. Jej atrybutem jest pole zawierające długość boku. Przyjmuje się, że długość podawana jest w jednostkach niemianowanych.

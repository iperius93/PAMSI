Detailed Description

Modeluje listę figur geometrycznych typu kwadrat i koło.

Klasa modeluje listę figur geometrycznych dwóch typów, tzn. kwadratów i kół. W wewnętrznej budowie klasy istnieją tylko jedna wspólna lista tych obiektów. Nie jest ona bezpośrednio widoczne poprzez interfejs publiczny klasy.

Wspomniany interfejs klasy umożliwia dodawanie nowych figur typu kwadrat lub koło. Pozwala również na wyświetlenie informacji o polach wszystkich figur, które zostały dodane do wcześniej wspomnianej listy.
